JavaScript Features:
- Dark mode toggle appears on the main pages and uses localStorage so the selected theme persists across pages.
- Career page uses IntersectionObserver to reveal timeline cards as they enter the viewport.
- Contact page uses the native Constraint Validation API with checkValidity and was-validated to validate fields.
- Contact form builds a mailto link dynamically so the visitor’s email client opens with subject and body prefilled.
- Projects page uses data-filter and data-category attributes to filter cards without inline onclick handlers.
- Projects filter buttons update aria-pressed for better accessibility.
- Finale page dynamically generates animated flames based on viewport width.
- Finale page uses an HTML5 canvas particle system to create explosions.
- Finale sequence is triggered when the video ends or when the user clicks “Skip to finale”.
- Finale page displays a HIRE ME overlay with CSS pop animation and floating music note elements.
- JavaScript avoids inline event handlers and uses addEventListener for cleaner behaviour.
